Disable Right Click For WP prevents right click on Wordpress site.

== Description ==
This plugin is used to disable right click on website to prevent copy content save image etc, It will stop the keyboard shortcut keys such as Ctrl A,X,C,V will be disabled by the plugin<br><br>
This Plugin is for one who wants to protect his online content from theft such as post text and images.  

=== Features ===

* Prevent Right Click.
* prevent Text Selection.
* This is the lightWeight plugin.
* Disable CTRL A,C,X,V for windows.
* No Negative side effects on SEO.
* Disable image drag/drop and save.


== Installation ==
* Go to the Add New plugins screen in your WordPress Dashboard
* Click the upload tab
* Browse for the plugin file (wp-prevent-right-click-content-copy) on wordpress.org
* Click \"Install Now\" and then hit the activate button.
